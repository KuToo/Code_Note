#### vim
- 三种工作模式：
    1.命令模式：默认进入vim的工作模式。该模式下，用户所有输入均被当成命令。
    2.编辑模式：编写模式，写代码、文本模式...
    3.末行模式：在命令模式下，按":"进入末行模式，该模式下，用户所有输入均被当成末行命令。
- 复制
    yy:复制光标所在行
    n(数字)yy：复制光标所在往后n行（包含本行）
    p:粘贴至光标下一行
    P:粘贴至光标上一行
    yw:将光标放置于单词的首字母上，复制一个单词
    区域复制：将光标放置于待复制区域的首字母上，按"v",使用hjkl移动光标，选择复制区域
- 剪切
    dd:剪切光标所在行
    n(数字)dd：剪切光标所在往后n行（包含本行）
    p:粘贴至光标下一行
    P:粘贴至光标上一行
    dw:将光标放置于单词的首字母上，剪切一个单词
    区域剪切：将光标放置于待剪切区域的首字母上，按"v",使用hjkl移动光标，选择剪切区域
    d0:从光标位置删除到行首
    d$(D):从光标位置删除到行尾
- 字符操作
    x:删除光标选中字符
    r:贯标选中字符，按“r”,按目标字符，进行字符替换
- 跳转到指定行
    命令模式下：
        gg:第一行
        GG:最后一行
        nG:跳转到第n行
        vim hello.go +56 打开文件的同时跳转到56行
        gg=G 自动排版
    末行模式下：
    :n+回车 跳转到第n行
- 查找
    想象一个关键词查找，先输入 "/"(显示在末行的位置)，再输入需要查找的单词
    查找一个已经看到的单词，将光标放在该单词任意一个字符，“ * ” 向后找，“ # ”向前找，n代表下一个
- 撤销 u
- 反撤销 ctrl+r    
- 替换--末行模式下
    单行替换
        :s/旧单词/新单词 有多个替换第一个
        :s/旧单词/新单词/g 有多个全部替换
    通篇替换
        :%s/旧单词/新单词 每一行有多个替换第一个
        :%s/旧单词/新单词/g 每一行有多个全部替换
    指定区域替换
        :n,ms/旧单词/新单词 替换n行到m行,每一行有多个替换第一个
        :n,ms/旧单词/新单词/g 替换n行到m行,每一行有多个全部替换

#### 远程登录服务器
- ssh
    1.查看是否安装ssh：sudo aptitude show ssh
    2.远程登录：ssh -l username ip(或者ssh -l username@ip) --输入密码
    3,退出远程登录：exit
- linux远程传输文件（目录）
    scp -r 源文件 远端username@ip:目标目录
